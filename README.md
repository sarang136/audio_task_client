# Audio Streaming App

A simple React + Vite application that streams and plays audio files in real time.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

The application will run on:

```text
http://localhost:5173
```

## Features

- Real-time audio streaming
- Dynamic audio rendering
- React Hooks
- Built with Vite

## Tech Stack

- React
- Vite
- JavaScript
- HTML5 Audio
